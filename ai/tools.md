# AI Tools Reference

## Claude Code Skills

### Global skills (`~/.claude/skills/`)
| Skill | Purpose |
|-------|---------|
| `gacp` | Git add / commit / push workflow |
| `update-config` | Modify `settings.json` hooks, permissions, env vars |
| `keybindings-help` | Customize `~/.claude/keybindings.json` |
| `simplify` | Review changed code for quality and efficiency |
| `less-permission-prompts` | Add allowlist entries to reduce permission prompts |
| `architecture` | Create / evaluate architecture decision records |
| `architecture-blueprint-generator` | Generate full architectural documentation |
| `frontend-design` | Build production-grade frontend UI |
| `prd` | Generate Product Requirements Documents |
| `system-design` | Design systems, services, and APIs |
| `skill-creator` | Create and optimize skills |
| `playwright-cli` | Automate browser interactions and Playwright tests |
| `claude-api` | Build / debug Claude API and Anthropic SDK apps |
| `security-review` | Security review of branch changes |
| `review` | Pull request review |
| `init` | Initialize a new `CLAUDE.md` file |
