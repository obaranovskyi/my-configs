# Snippets

## AI

| Name | Keyword | Description |
|------|---------|-------------|
| Reverse Prompt | `;;air` | Asks AI to reconstruct the ideal prompt based on the current result. |
| Simplified Reverse Prompt | `;;aisr` | Simplified version of the Reverse Prompt snippet that focuses on reconstructing the ideal prompt based on the current result. |

## English

| Name | Keyword | Description |
|------|---------|-------------|
| Review & provide a more natural way to say it | `;;eni` | Reviews clipboard text for mistakes and suggests a more natural phrasing. |
| 10 natural alternatives | `;;eno` | Generates 10 natural alternative phrasings for the clipboard text. |
| Grammar check | `;;eng` | Fixes grammar mistakes in the clipboard text and explains the changes. |

## My

| Name | Keyword | Description |
|------|---------|-------------|
| Email address | `;;em` | Expands to my personal email address. |

## Git

| Name | Keyword | Description |
|------|---------|-------------|
| Push All Changes | `;;gp` | Stages all changes, commits with a prompted message, and pushes. |
| Ai Push All Changes | `;;aigp` | Stages all changes, generates a commit message with AI based on the diff, commits, and pushes. |
